Need help?

Support, bug reports, and feature requests are tracked publicly on GitHub.

  1. Visit github.com/andybrucenet/chart-finder/issues.
  2. Search for an existing issue that matches your question.
  3. If nothing fits, open a new issue that includes reproduction steps and logs.

Automations can also file tickets through the GitHub Issues REST API against the andybrucenet/chart-finder repository.
